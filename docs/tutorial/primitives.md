# Primitives

As explained in the [Types](types) section, Maize doesn't have any of the
"normal" primitives.

They still exist, but they're implemented in the Maize standard library.

```haskell
five = 5

threeNinths = 3 / 9

isFalse = not True

name = "Maize"
```
